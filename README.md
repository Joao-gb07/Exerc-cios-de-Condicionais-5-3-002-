**Exercícios de Condicionais em Java**

**Exercício 1: Ler a idade de uma pessoa e determinar a sua categoria: Criança (0-12 anos), Adolescente (13-17 anos) ou Adulto (18 anos ou mais).**
Saída Esperada:

Se a idade lida for 10, a saída será: Criança.
Se a idade lida for 15, a saída será: Adolescente.
Se a idade lida for 25, a saída será: Adulto.

**Exercício 2: Ler a temperatura em Celsius e escrever se está frio (menor que 15), ameno (entre 15 e 25) ou quente (maior que 25).**
            
Saída Esperada:

Se a temperatura lida for 10, a saída será: Frio.
Se a temperatura lida for 20, a saída será: Ameno.
Se a temperatura lida for 30, a saída será: Quente.

**Exercício 3: Ler um número de 1 a 12 e escrever o mês correspondente. Caso o número não esteja nesse intervalo, exibir uma mensagem de erro.**
            
Saída Esperada:

Se o número lido for 3, a saída será: Março.
Se o número lido for 15, a saída será: Número inválido. Digite um número de 1 a 12.

**Exercício 4: Ler um valor e verificar se é um número de dois dígitos.**
        
Saída Esperada:

Se o número lido for 45, a saída será: O número é de dois dígitos.
Se o número lido for 123, a saída será: O número NÃO é de dois dígitos

**Exercício 5: Ler um número de 1 a 7 e escrever o dia da semana correspondente.**
            
Saída Esperada:

Se o número lido for 3, a saída será: Terça-feira.
Se o número lido for 8, a saída será: Número inválido. Digite um número de 1 a 7.

**Exercício 6: Ler um valor e escrever a moeda correspondente (1 para Real, 2 para Dólar, 3 para Euro).**
            
Saída Esperada:

Se o número lido for 2, a saída será: Dólar.
Se o número lido for 4, a saída será: Moeda inválida. Digite um número de 1 a 3

**Exercício 7: Ler um número correspondente a um mês (1 a 12) e escrever a quantidade de dias do mês (considerando fevereiro com 28 dias).**
            
Saída Esperada:

Se o número lido for 2, a saída será: 28 dias.
Se o número lido for 9, a saída será: 30 dias.
Se o número lido for 13, a saída será: Mês inválido. Digite um número de 1 a 12.
